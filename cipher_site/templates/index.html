Substitution Cipher Decrypter

Paste a cipher text and pick a method. We’ll try to crack it and format the result into words.

How it works

The monoalphabetic solver uses a genetic algorithm to explore possible keys, then a hill-climber to refine the best one. Decryptions are scored with n-gram statistics (bigrams, trigrams, quadgrams) and then segmented into words using word frequencies.

Built with your existing solver code. Ready for deployment (Flask).
